Introducción

¿Qué es OpenStreetMap?

OpenStreetMap (OSM) es un mapa callejero gratuito de todo el mundo, creado por una siempre creciente comunidad de entusiastas de los mapas

Utilizando el mismo concepto de Wikipedia, es creado por los ciudadanos para que lo use cualquiera como vos incluso comercialmente

Esperamos que encuentres OpenStreetMap útil e interesante...

Comenzar ahora Aprende más

¿Cómo empezar a contribuir?

  • * Úsalo en el estudio, trabajo, salidas, pasatiempos
  • * Divulga de qué trata OpenStreetMap en tu entorno
  • * Pide a las instituciones públicas y/o privadas que liberen sus datos bajo la licencia CC BY-SA (opendata)
  • * Ayúdanos reportando errores y/o omisiones (más info)
  • * Diviértete editando el mapa de tu zona vos mism@...

Usos OpenStreetMap

Enrutamiento (Routing)

Open Source Routing Machine (OSRM) es una aplicación web que te ayuda a planificar rutas cortas para usarlas mientras conduces automóvil. Combina sofisticada ingeniería y lo mejor del software open source usando datos del mapa OpenStreetMap (OSM)

Existe muchísimos más aplicaicones según tu actividad... si manejas bicicleta en la ciudad, en la montaña (mtb), acampas (hiking) o haces senderismo (trekking) hay un mapa especial para vos

Aplicaciones Android/iOS

Hay una gran variedad de software móvil para la visualización de OpenStreetMap que no usan tu plan de datos en los teléfonos y otros dispositivos móviles. Cada vez más aplicaciones están permitiendo contribución a OpenStreetMap también

Aplicaciones como OSMAnd, NavFree, OSMTracker, OSMdroid, MapQuest para Android. PushPin, Poi+, MapQuest, GPSTracks para iOS... Existen muchísimas aplacaciones esperándote! ;-)

Encuentra más

Mapas interactivos

Tanto si eres un periodista, diseñador web, investigador o cartógrafo experimentado, TileMill es el estudio de diseño que necesita para crear mapas interactivos sorprendentes

Tienes información tabulada geolocalizada en formato CSV utiliza GeoJSON.io y hospeda tus datos dentro de GitHub sin costo. Quieres crear y publicar rápidamente utiliza UMap o SlippyMap. Depende qué desees hacer hay una aplicación que puede satisfacerte
